# Як застосувати міграції

## Способи застосування міграцій


### 1. При старті додатку з ключем migrate (рекомендовано)

Міграції застосовуються при запуску додатку з ключем migrate

```bash
booking-meeting-rooms.exe migrate
```
Створює базу даних автоматично, якщо її немає
Застосовує міграції
Додає seed дані автоматично


### 2. Через EF Core CLI (потрібні коди)

```bash
dotnet ef database update
```
Не створює базу даних автоматично (якщо бази немає, буде помилка)
Не додає seed дані (користувачі та кімнати не будуть створені автоматично)


## Важливо
- Перед застосуванням міграцій переконайтеся, що PostgreSQL запущений
- Перевірте рядок підключення в `appsettings.json`
